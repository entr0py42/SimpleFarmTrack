# Commands
| Command                         | Description   |
| :-------------------------------|:--------------|
| `/startsft`                     | Starts the timer and loot tracking
| `/endsft`                       | Ends the process and log the results in the chat.
| `/pausesft`                     | Pause the timer and loot tracking
| `/resumesft`                    | Resume the timer and loot tracking
